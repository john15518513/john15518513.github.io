Work

Smart Home Application is an IoT application which allows users to remotely control their electronic appliances and monitor the condition of their home. We incorporate the idea of wireless mesh network with our application which significantly increases the scalability and reliability of our project. In this work, I was responsible for developing the communication among clients, server, and IoT devices using Amazon IoT API, Amazon IoT Python SDK, and Amazon IoT NodeJS SDK. I also built wireless mesh network for each device from scratch.

Keyword: NodeJS, Socket.io, Amazon IoT, Amazon EC2, Wireless Mesh Network, Python, Shell Script

Point Location Graphical Applets is a Java applet that demonstrates Kirkpatrick’s algorithm that locates any point on planner graph in O(lgn). Users could generate their own planar graph and the applet will show the point locating run-time procedure on graphical interface. In this project, I am responsible for implementing polygon triangulation algorithm and user interface.

project source code

Keyword: Algorithm, Data Structures, Computational Geomtry, GUI, Java

Wireless Energy Performance Model is an energy performance monitoring system designed for solar decathlon house. Users could keep track of their expenses on gas, water, and electricity usage of their home. We also visualize ambient stats such as temperature, humidity, and light data on our website. In this project, I am responsible for developing application on Telosb in nesC, developing application on Arduino in C, developing application on RaspberryPi in Python, and developing server side program in Java.

Keyword: TinyOS, nesC, Embedded Systems, Python, Java, C.

Portfolio Selection Strategy Developed a mutual fund investment management system to assist users in selecting portfolio. Designed fitness function of genetic algorithm and ran on over one million of input data scraped from financial websites. Results show our strategy is positive for average case, and could still win someone who picks portfolio randomly for worst case.

Keyword: Maching Learning, Genetic Algorithm, Data Scraping, C++.

Research & Development Conducted research on emerging memory and hybrid memory system. Built a C++ trace-driven embedded DRAM simulation tool based on the architecture of two open source C/C++ simulators – DineroIV and DRAMSimII. The simulator was customized based on United Microelectronics Corporation specs and the accuracy was validated by UMC. Proposed Asymmetric Bank Design for embedded DRAM cache which significantly reduced data access latency by 16.9% and improved energy efficiency by an average of 46.4%.

Keyword: Computer Architecture, Memory Systems, Embedded Systems, Simulator, C++.
